10.3 - Búsqueda de archivos y contenido

find locate grep

Comando find - Búsqueda avanzada por criterios

find busca archivos y carpetas por nombre o tipo recorriendo directorios. Aquí verás lo básico para usarlo a diario.

Sintaxis básica

Terminal
$ find [ruta] [opciones] [expresión]

Lo básico que necesitas

-name "patrón" por nombre (sensibles a mayúsculas)
-iname "patrón" por nombre (sin distinguir mayúsculas)
-type f archivos
-type d directorios

Ejemplos rápidos

usuario@mint:~$ find . -name "*.txt"
./notas.txt
./docs/lista.txt

usuario@mint:~$ find . -iname "foto*"
./Fotos/foto01.jpg

usuario@mint:~$ find . -type d
.
./Fotos
./docs

Comando locate - Búsqueda rápida

locate encuentra rutas por nombre usando una base de datos del sistema. Es muy rápido para búsquedas simples por nombre.

Nota importante: locate utiliza una base de datos que se actualiza diariamente. Para actualizar manualmente: sudo updatedb

Sintaxis

Sintaxis locate
$ locate [opciones] patrón
Opciones comunes:
-i ignorar mayúsculas -l N limitar resultados

Ejemplos prácticos

usuario@mint:~$ locate firefox
/usr/bin/firefox
/usr/share/applications/firefox.desktop

usuario@mint:~$ locate -i -l 5 "*.pdf"
/home/usuario/Documentos/manual.pdf
/home/usuario/Descargas/guia.pdf
/home/usuario/Esquemas/apuntes.pdf
/home/usuario/proyectos/info.pdf
/home/usuario/Escritorio/presentacion.pdf

Comando grep - Buscar texto dentro de archivos

grep busca texto dentro de archivos o de la salida de otros comandos. Ideal para encontrar palabras o líneas rápidamente.

Sintaxis básica

Sintaxis grep
$ grep [opciones] "patrón" archivo(s)
$ comando | grep [opciones] "patrón"

Opciones básicas

-i ignorar mayúsculas -n mostrar número de línea -r buscar en subcarpetas -c contar coincidencias

Ejemplos rápidos

usuario@mint:~$ grep "Linux" notas.txt
Aprendiendo Linux paso a paso

usuario@mint:~$ grep -n -i "error" app.log
12:ERROR: conexión fallida
usuario@mint:~$ grep -r "TODO" .
./src/main.py:# TODO: refactorizar
./docs/readme.md:TODO: agregar ejemplos

Práctica guiada (3 ejercicios rápidos)

Pon en práctica lo esencial de find, grep y locate sin riesgos.

Preparación: Abre una terminal y ve a tu carpeta personal con cd ~
Ejercicio 1: Prepara un mini entorno
  1. Crea una carpeta y 3 archivos de prueba:
    mkdir -p busqueda_demo/sub
    echo "Hola Linux" > busqueda_demo/notas.txt
    echo "Configuración" > busqueda_demo/config.conf
    echo "TODO: aprender grep" > busqueda_demo/sub/tareas.txt
    Salida esperada
    Estos comandos no muestran salida si se ejecutan correctamente.
Ejercicio 2: find básico
  1. Lista archivos: find busqueda_demo -type f
    Salida esperada (ejemplo)
    busqueda_demo/notas.txt
busqueda_demo/config.conf
busqueda_demo/sub/tareas.txt
  2. Busca por patrón: find busqueda_demo -name "*.txt"
    Salida esperada
    busqueda_demo/notas.txt
busqueda_demo/sub/tareas.txt
  3. Lista solo carpetas: find busqueda_demo -type d
    Salida esperada (ejemplo)
    busqueda_demo
busqueda_demo/sub
Ejercicio 3: grep básico
  1. Buscar una palabra: grep "Linux" busqueda_demo/*.txt
    Salida esperada
    busqueda_demo/notas.txt:Hola Linux
  2. Buscar recursivo (sin distinguir mayúsculas): grep -r -i "todo" busqueda_demo
    Salida esperada
    busqueda_demo/sub/tareas.txt:TODO: aprender grep
  3. Contar coincidencias en un archivo: grep -c "o" busqueda_demo/notas.txt
    Salida esperada
    1
Extra (opcional): si tienes la base actualizada (sudo updatedb), prueba locate -i -l 5 pdf.
¡Fantástico! Ahora sabes cómo encontrar archivos por nombre, ubicación y contenido. Estas herramientas te ayudarán mucho en tu trabajo diario con Linux.
Recuerda: find = por nombre/tipo, grep = por texto, locate = rápido por nombre (con updatedb).
