Herramientas de personalización

Cap. 11 — Linux Mint (Cinnamon)

Temas Iconos Fondos Panel Applets Desklets Extensiones

¿Dónde se personaliza?

Todo está en Configuración del sistema (Cinnamon).

  • Temas: Cambia apariencia de ventanas, controles, íconos y cursor.
  • Fondo de escritorio: Elige o agrega tus imágenes.
  • Applets y Panel: Elementos en la barra (reloj, red, etc.).
  • Desklets: Widgets en el escritorio (reloj, notas, monitor).
  • Extensiones: Funciones extra para el escritorio.
Abre el menú, escribe “Temas” o “Configuración” y entra directo al módulo.

Temas e iconos

Usa Temas para cambiar:

  • Controles (botones y menús), Ventanas, Iconos y Cursor.
  1. Abre Temas.
  2. Prueba Mint-Y o Mint-Y-Dark (oscuro).
  3. Cambia Iconos a Mint-Y o Mint-L.
  4. Botón Añadir/Descargar para más opciones.
Puedes mezclar: tema oscuro + iconos claros, etc.

Fondos y presentación

En Fondo de escritorio:

  1. Elige una imagen de Mint o añade las tuyas (botón +).
  2. Activa Presentación para cambiar automáticamente.
  3. Ajusta Rellenar/Adaptar/Centrar según tu preferencia.
La presentación puede cambiar cada 10 min, 30 min, etc.

Panel y applets

El Panel es la barra inferior. Puedes:

  • Moverlo: clic derecho → Configuración del panel → Posición.
  • Añadir applets: clic derecho → Añadir applets al panel.
  • Applets útiles: Reloj mejorado, Monitor del sistema, Clima.
Si rompes algo, usa “Restaurar configuración del panel”.

Desklets y extensiones

Desklets: widgets en el escritorio (reloj, notas, fotograma).

Extensiones: agregan funciones (ventanas, espacios de trabajo, atajos).

  1. Abre Desklets o Extensiones.
  2. Instala desde Descargar y actívalos.
  3. Configura opciones con el ícono de ajustes.
Prefiere elementos populares y mantenidos para evitar fallos.

Variety (fondos automáticos)

Cambia el fondo automáticamente con imágenes locales o en línea.

Instalación: sudo apt install variety
  1. Abre Variety y elige tus carpetas de imágenes.
  2. Define el intervalo (por ejemplo, cada 15 minutos).
  3. Activa Iniciar con el sistema si te gusta el resultado.
Puedes pausar el cambio cuando estés trabajando con colores fijos.

Plank (dock simple)

Añade un dock sencillo para tus apps favoritas.

Instalación: sudo apt install plank
  1. Ejecuta Plank desde el menú.
  2. Arrastra apps al dock para fijarlas.
  3. Preferencias → posición, tamaño y tema del dock.
Para iniciar con el sistema: Aplicaciones al inicio → añadir plank.

Papirus (tema de iconos)

Paquete de iconos moderno y muy popular, con variantes claras y oscuras.

Instalación: sudo apt install papirus-icon-theme
  1. Abre Temas → sección Iconos.
  2. Elige Papirus o Papirus-Dark.
  3. Combínalo con Mint-Y-Dark para un look uniforme.
Disponible en los repos de Mint/Ubuntu.

Práctica guiada (5 minutos)

Tres cambios rápidos para dejar Mint a tu estilo.

1) Tema oscuro y iconos
  1. Abre Temas → selecciona Mint-Y-Dark.
  2. En Iconos, elige Mint-Y (o el que prefieras).
Resultado esperado
Ventanas y menús con fondo oscuro.
Iconos del sistema actualizados al conjunto elegido.
2) Fondo con presentación
  1. Abre Fondo de escritorio y elige una carpeta de imágenes.
  2. Activa Presentación y define cada 10–30 min.
Resultado esperado
El fondo cambia automáticamente con tus imágenes.
Se muestra bien en todas las pantallas.
3) Panel con applet útil
  1. Clic derecho en el panel → Añadir applets.
  2. Instala/activa Monitor del sistema o Clima.
  3. Opcional: mueve el panel a la parte superior.
Resultado esperado
Ves el applet elegido en el panel (CPU/red/tiempo).
El panel aparece donde lo configuraste.
Listo: tema, iconos, fondo y panel personalizados.
