Monitoreo de recursos

Herramientas sencillas para ver CPU, memoria, red y disco, válidas para cualquier distribución y entorno de escritorio.

btop htop df free uptime nethogs iotop

Monitores gráficos (opcional)

Si prefieres una interfaz gráfica, cada entorno de escritorio suele tener su propio monitor de sistema:

  • GNOME: Monitor del sistema (gnome-system-monitor)
  • KDE: KSysGuard (ksysguard)
  • LXDE/LXQt: LXTask (lxtask)
  • Xfce: Monitor del sistema (xfce4-taskmanager)

Busca "monitor" o "task manager" en el menú de tu sistema. Si no tienes ninguno, puedes instalar el de tu entorno o usar solo terminal.

Los comandos de terminal funcionan en cualquier distribución y escritorio.

Discos (SMART y particiones)

Para revisar discos y particiones gráficamente, puedes usar:

  • GNOME: Discos (gnome-disk-utility)
  • KDE: KDE Partition Manager (partitionmanager)
  • O usa terminal: lsblk, df -h, smartctl
Para ver el estado SMART por terminal: sudo smartctl -a /dev/sdX (requiere smartmontools).

btop (recomendado)

Vista moderna de CPU, memoria, procesos, red y disco.

usuario@mint:~$ sudo apt install -y btop
usuario@mint:~$ btop # Teclas: q salir, F2 opciones, 1-4 paneles
Resultado esperado
Se abre la interfaz de btop con gráficos y listas de procesos.

htop (alternativa clásica)

Más liviano y muy conocido.

usuario@mint:~$ sudo apt install -y htop
usuario@mint:~$ htop # F6 ordenar, F3 buscar, q salir
Resultado esperado
Aparece htop con procesos y barras de CPU/memoria.

Opcional: red y disco en vivo

usuario@mint:~$ sudo apt install -y nethogs iotop # Requieren permisos de administrador
usuario@mint:~$ sudo nethogs # Ver consumo de red por proceso
usuario@mint:~$ sudo iotop # Ver lectura/escritura de disco por proceso

Práctica guiada (5 minutos)

Tres comprobaciones rápidas de tu sistema.

1) Instala y abre btop
usuario@mint:~$ sudo apt install -y btop && btop
Resultado esperado
Se abre btop mostrando CPU, memoria y procesos.
2) Revisa el disco
usuario@mint:~$ df -h
Salida esperada
/dev/sda2       100G   45G   50G  48% /
3) Revisa la memoria
usuario@mint:~$ free -h
Salida esperada
Mem:   15Gi total, 3.1Gi usados, 12Gi disponibles
Listo: ya sabes ver rápidamente salud y uso de tu equipo.
