Esenciales del Sistema

Antes de empezar, asegúrate de tener los cimientos: curl, git y compadres.

usuario@mint:~$ sudo apt update && sudo apt install -y curl git build-essential # Actualiza repositorios e instala utilidades base

Antigravity AI

Tu asistente de programación. Instálalo como extensión en tu editor para tener un pair programmer 24/7.

$ sudo mkdir -p /etc/apt/keyrings
$ curl -fsSL https://us-central1-apt.pkg.dev/doc/repo-signing-key.gpg | sudo gpg --dearmor --yes -o /etc/apt/keyrings/antigravity-repo-key.gpg
$ echo "deb [signed-by=/etc/apt/keyrings/antigravity-repo-key.gpg] https://us-central1-apt.pkg.dev/projects/antigravity-auto-updater-dev/ antigravity-debian main" | sudo tee /etc/apt/sources.list.d/antigravity.list > /dev/null
$ sudo apt update
$ sudo apt install antigravity

Visual Studio Code

El editor de código más versátil. Instalación limpia vía Flatpak.

$ flatpak install flathub com.visualstudio.code

También disponible en el Gestor de Software como "Code".

Git

Ctrl+Z para la vida real. Configura tu identidad:

$ sudo apt install -y git
$ git config --global user.name "Tu Nombre"
No olvides user.email.

Python 3

Lenguaje universal. Usa entornos virtuales siempre.

$ sudo apt install -y python3-venv python3-pip
$ python3 -m venv .venv

Node.js (NVM)

Usa nvm, no el de los repositorios.

$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
$ nvm install --lts

Docker

Contenedores para despliegue y desarrollo limpio.

$ sudo apt install docker.io
$ sudo usermod -aG docker $USER

Data & Testing

sudo apt install mariadb-server
