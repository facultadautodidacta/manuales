Cap. 12 — Lo esencial hoy y cómo instalarlo en Linux Mint

Git VS Code Node.js (nvm) Python Docker SQLite

Esenciales del sistema

Primero, algunas utilidades que todo dev usa:

  • curl (descargas en terminal), git (control de versiones), build-essential (compiladores).
usuario@mint:~$ sudo apt update && sudo apt install -y curl git build-essential # Actualiza e instala utilidades básicas
Usa el Gestor de software si prefieres interfaz gráfica.

Git

Instala y configura tu identidad:

usuario@mint:~$ sudo apt install -y git # Instala Git
usuario@mint:~$ git --version
git version 2.x.x

usuario@mint:~$ git config --global user.name "Tu Nombre" usuario@mint:~$ git config --global user.email "tu@correo.com" # Configura tu identidad
Salida esperada
git version 2.x.x

Visual Studio Code

Instalación sencilla por Flatpak (recomendado por facilidad):

sudo apt install -y flatpak
flatpak install -y flathub com.visualstudio.code
También puedes instalarlo desde Gestor de software buscando “Visual Studio Code”.

Node.js con nvm

Gestiona versiones de Node fácilmente con nvm:

usuario@mint:~$ curl -fsSL https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash # Instala nvm
# Cierra y abre la terminal nuevamente
usuario@mint:~$ nvm install --lts usuario@mint:~$ node -v
v18.x.x
usuario@mint:~$ npm -v
9.x.x
Salida esperada
v18.x.x
9.x.x

Python (venv + pipx)

Entornos aislados y herramientas globales limpias:

sudo apt install -y python3 python3-venv python3-pip pipx
python3 --version
# Crear entorno para un proyecto
python3 -m venv .venv
source .venv/bin/activate
pip install requests
# Herramientas globales sin ensuciar el sistema
pipx ensurepath
pipx install black
Salida esperada
Python 3.x.x
(.venv) $

Docker (rápido)

Versión simple desde repositorios de Mint/Ubuntu:

sudo apt install -y docker.io docker-compose-plugin
sudo usermod -aG docker $USER
# Cierra sesión y vuelve a entrar para aplicar el grupo
docker --version
Salida esperada
Docker version 24.x.x, build ...
Para producción, considera la guía oficial de Docker (repositorio upstream).

Base de datos ligera: SQLite

Ideal para empezar y pruebas locales:

sudo apt install -y sqlite3
sqlite3 --version
Salida esperada
3.4x.x 20xx-xx-xx ...
¿Necesitas servidor? Prueba sudo apt install postgresql y usa psql.

Pila LAMP (Apache, MariaDB, PHP)

Ideal para desarrollo web clásico en PHP.

usuario@mint:~$ sudo apt install -y apache2 mariadb-server php libapache2-mod-php php-mysql # Instala servidor web, base de datos y PHP
usuario@mint:~$ sudo systemctl enable --now apache2 mariadb # Habilita e inicia servicios

Prueba rápida de PHP:

usuario@mint:~$ echo "<?php phpinfo(); ?>" | sudo tee /var/www/html/info.php >/dev/null # Crea /info.php
usuario@mint:~$ xdg-open http://localhost/info.php # Abre la página en tu navegador
Para asegurar MariaDB: sudo mysql_secure_installation.
