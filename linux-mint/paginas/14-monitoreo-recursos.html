Monitoreo de recursos

Herramientas fáciles para ver CPU, memoria, red y disco

System Monitor Discos btop htop df free

Monitor del sistema (gráfico)

Abre el Monitor del sistema desde el menú (o instala si no lo tienes):

usuario@mint:~$ sudo apt install -y gnome-system-monitor # CPU, memoria, procesos y red con interfaz gráfica
  • Ordena por CPU o Memoria para detectar procesos pesados.
  • Cierra procesos colgados desde la pestaña de Procesos.

Discos (SMART y particiones)

La app Discos muestra información de unidades y estado SMART.

usuario@mint:~$ sudo apt install -y gnome-disk-utility # Revisa salud del disco y gestiona montajes
Si ves sectores reasignados o errores SMART, haz copia de seguridad.

btop (recomendado)

Vista moderna de CPU, memoria, procesos, red y disco.

usuario@mint:~$ sudo apt install -y btop
usuario@mint:~$ btop # Teclas: q salir, F2 opciones, 1-4 paneles
Resultado esperado
Se abre la interfaz de btop con gráficos y listas de procesos.

htop (alternativa clásica)

Más liviano y muy conocido.

usuario@mint:~$ sudo apt install -y htop
usuario@mint:~$ htop # F6 ordenar, F3 buscar, q salir
Resultado esperado
Aparece htop con procesos y barras de CPU/memoria.

Opcional: red y disco en vivo

usuario@mint:~$ sudo apt install -y nethogs iotop # Requieren permisos de administrador
usuario@mint:~$ sudo nethogs # Ver consumo de red por proceso
usuario@mint:~$ sudo iotop # Ver lectura/escritura de disco por proceso

Práctica guiada (5 minutos)

Tres comprobaciones rápidas de tu sistema.

1) Instala y abre btop
usuario@mint:~$ sudo apt install -y btop && btop
Resultado esperado
Se abre btop mostrando CPU, memoria y procesos.
2) Revisa el disco
usuario@mint:~$ df -h
Salida esperada
/dev/sda2       100G   45G   50G  48% /
3) Revisa la memoria
usuario@mint:~$ free -h
Salida esperada
Mem:   15Gi total, 3.1Gi usados, 12Gi disponibles
Listo: ya sabes ver rápidamente salud y uso de tu equipo.
